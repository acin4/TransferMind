# How to run backend so cluster analysis works properly

In terminal:

1. from root folder -> cd backend
2. source .venv/bin/activate
3. python -m pip install -r requirements.txt
4. npm run dev

# How to run backend so cluster analysis works properly( on WINDOWS )

     In terminal:

1. cd backend
2. .\venv\Scripts\Activate.ps1
3. $env:TRANSFERMIND_PYTHON_BIN=".\venv\Scripts\python.exe"
4. npm run dev
